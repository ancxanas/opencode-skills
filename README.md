## Usage

1. Install [OpenCode](https://opencode.ai)
2. Clone this repo into `~/.config/opencode/`:

   ```bash
   git clone https://github.com/ancxanas/opencode-skills.git ~/.config/opencode
   ```

3. Skills are auto-detected by OpenCode based on context.

## Maintenance

A daily GitHub Action checks high-drift skills (framework/package version-specific) against latest registry versions and opens `skill-stale` issues when major version mismatches are detected.
