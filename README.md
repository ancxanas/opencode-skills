## Usage

1. Install [OpenCode](https://opencode.ai)
2. Clone this repo into `~/.config/opencode/`:

   ```bash
   git clone https://github.com/ancxanas/opencode-skills.git ~/.config/opencode
   ```

3. Skills are auto-detected by OpenCode based on context.
